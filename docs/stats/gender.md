# 性別

![性別グラフ](../../assets/images/stats_gender.png)

## データ仕様
| 項目名 | 説明 |
| ---- | ---- |
| データ項目 | 性別（case_sex）、症例数 |
| データ形式 | 性別は文字列（例: male, unknown）、人数は正の整数 |

## グラフ仕様
| 項目名 | 説明 |
| ---- | ---- |
| グラフの種類 | 円グラフ |
| 凡例 | 男性（male）、女性（female）、その他（other）、不明（unknown） |
| ツールチップ | ユーザーがセグメントにマウスオーバーした際、性別と対応する人数を表示 |
